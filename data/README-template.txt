M.A.1 & M.A.2 Inspection Templates
====================================

Template files created:
- data/template-M.A-1.xlsx (for license type M.A.1)
- data/template-M.A-2.xlsx (for license type M.A.2)
- data/template-M.A-1.csv (CSV version)
- data/template-M.A-2.csv (CSV version)

To upload these in the app:
1. Open the app
2. Click "ટેમ્પલેટ મેનેજર" from sidebar
3. Click "ફાઇલ પસંદ કરો" 
4. Select the appropriate xlsx/csv file
5. Preview and confirm upload
